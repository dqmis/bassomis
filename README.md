## bassomis music blog

---

This is a source code of music blog [bassomis](https://bassomis.lt). Code is separated into three branches:
* master - Code for Heroku server
* backend - Node.js and express code
* frontend - Angular 2 code

---

Built with:

* Angular 2
* Node.js
* Express.js
* MongoDB
